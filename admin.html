Export & Deploy
Download your config and push to GitHub to go live
How publishing works:
1. Edit your content above and click Save Changes.
2. Click Download config.json below.
3. Place the file at data/config.json in your local repo.
4. git add data/config.json && git commit -m "Update site config" && git push
5. Railway redeploys in ~30 seconds. All visitors will see your changes.
